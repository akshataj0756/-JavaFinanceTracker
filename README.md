# Java Finance Tracker

## Overview
Console-based personal finance management system developed using Java and MySQL.

## Tech Stack
- Core Java
- JDBC
- MySQL

## Features
- User Login
- Add Income
- Add Expense
- Transaction History
- Category-wise Tracking
