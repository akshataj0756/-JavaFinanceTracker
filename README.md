# Java Finance Tracker

## Project Description

Java Finance Tracker is a console-based personal finance management application developed using Java and MySQL. The application enables users to manage their financial transactions efficiently by providing features such as user registration, login, transaction management, filtering, and expense summary.

## Features

- User Registration
- User Login
- Forgot Password
- Add Transaction
- View All Transactions
- Update Transaction
- Delete Transaction
- Filter Transactions by Category
- View Expense Summary
- MySQL Database Integration
- DAO Architecture using JDBC
