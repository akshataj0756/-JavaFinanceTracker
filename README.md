 JavaFinanceTracker

•	Project Information
JavaFinanceTracker is a Java-based application developed to help individuals and organizations manage their financial transactions effectively. It provides a structured way to record, categorize, and monitor income and expenses. The application is designed with scalability in mind so that it can be extended to support databases, graphical interfaces, and additional features.

•	Project Overview
The primary goal of JavaFinanceTracker is to simplify financial tracking for different types of users. The project starts with a simple console-based implementation where users can add and manage transactions. As the project evolves, it can be enhanced to support multiple roles, including guests and administrators, providing different levels of functionality and control.

•	Features
- Add, view, and manage income and expenses
- Categorize transactions (e.g., food, travel, bills, salary, savings)
- Display current balance in real-time
- Provide transaction history for analysis
- Role-based access (User, Guest, Admin) for controlled functionality
- Extensible architecture for future integration with databases or UI frameworks

•	Role-Based Functionality

•	Guest
- Can view limited information such as a demo of how the application works
- Can explore sample financial data for understanding features
- Cannot add, modify, or delete transactions

•	User
- Can register and log in to the application
- Can add new income and expense records
- Can categorize and view transaction history
- Can calculate and view available balance
- Can update or delete their own transactions
- Has access to personalized financial tracking

•	Admin
- Can manage all users and their accounts
- Can view, update, or delete transactions across all users
- Can monitor categories and system-wide data
- Responsible for maintaining overall data integrity
- Can generate reports and manage backups

•	Future Enhancements
- Database integration for persistent storage
- Graphical user interface using JavaFX or Swing
- Data visualization with charts and graphs
- Export of reports in PDF or CSV formats
- Multi-user authentication and secure login system
